# RaceLap

[![lerna](https://img.shields.io/badge/maintained%20with-lerna-cc00ff.svg)](https://lerna.js.org/)

## Install

```bash
# install root deps
npm install

# install packages deps
npx lerna bootstrap
```

## Dev

```bash
npx lerna run start --stream
```
